# Homebrew Tap

Custom Homebrew formulas by badriram.

## Installation

```bash
brew tap badriram/tools
brew install markdown-preview
```

## Formulas

### markdown-preview

Generate PDF previews of Markdown files for macOS Finder Quick Look.

**Usage:**
```bash
markdown-preview /path/to/file.md
```

Creates a PDF next to the original file and opens it in Quick Look.
